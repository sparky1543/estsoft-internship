**사용자**

1. 삭제 사용자 수 → 탈퇴 사용자 수로 변경
    
    해당 지표는 Leave 상태를 참조하고 있어, 용어를 탈퇴로 수정
    
    탈퇴(Leave) 상태의 사용자는 30일 이후 계정 삭제(Delete) 로 전환되며, 이때 row가 갱신되어 지표 값이 감소
    
    일정 기간 이후 조회 시 추이 값이 변동되는 현상 발생 → 지표 집계 방식 논의 필요
    
    → leave + delete로 조회
    
2. 총 사용자 수 집계 기준 검토
    
    탈퇴 및 삭제 사용자를 포함할지 여부 결정
    
    포함 여부에 따라 지표명을 명확하게 수정 필요
    
    → 탈퇴 사용자는 제외하고 집계, 지표명 수정 필요
    
3. 지금 사용자 통계에 구독 관련 지표는 빠졌는데 상세 보고서에도 제외할건지 논의
    
    → 상세 보고서 내용 수정 예정
    

**구독**

1. 구독 통계의 도넛 차트들도 월 고정 값인지 확인
    
    월 고정이 아닐 경우, 월별 필터 적용시 두 달의 합계로 표시할 건지 여부 결정
    
2. 구독 상세보고서에서도 Team, Starter 추가 할건지 논의
3. 신규 구독자 추가 논의
